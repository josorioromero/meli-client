# Test Front End Mercado Libre - Jorge Osorio

- `git clone https://github.com/josorioromero/meli-client.git`

## RUN API
- `cd meli-client/server`
- `yarn` or `npm i`
- `yarn start` or `npm start`
- The API will start in `localhost:8000`

## RUN Client
- `cd meli-client`
- `yarn` or `npm i`
- `yarn start` or `npm start`
- The API will start in `localhost:3000`

## Run tests
- `yarn test` or `npm test` will run the tests suite

### STACK
- Client: React, Redux, SASS
- Server: NodeJS, Express
